# 💡 Быстрые подсказки для Selectel

## Выбор операционной системы

### ✅ ПРАВИЛЬНО (для нашего приложения):
- **Ubuntu 24.04 LTS 64-bit** (обычная версия)
- Без GPU оптимизации
- Без Docker (опционально, можно добавить позже)

### ❌ НЕ НУЖНО:
- ❌ Варианты с "GPU Driver 580" или "GPU Driver 535"
- ❌ Варианты с пометкой "GPU optimized"
- ❌ Docker не обязателен для начала

**Почему?** 
- Наше приложение (Node.js + React) не использует GPU
- GPU-оптимизированные варианты дороже и нам не нужны
- Docker можно установить позже, если понадобится

## Рекомендуемая конфигурация

### Минимальная (для тестирования):
- **RAM:** 2GB
- **CPU:** 2 ядра
- **Диск:** 20GB SSD
- **ОС:** Ubuntu 24.04 LTS (без GPU)

### Рекомендуемая (для продакшена):
- **RAM:** 4GB
- **CPU:** 4 ядра
- **Диск:** 40GB SSD
- **ОС:** Ubuntu 24.04 LTS (без GPU)

## Стоимость на Selectel

- **Минимальная:** ~300-500₽/месяц
- **Рекомендуемая:** ~600-1000₽/месяц

## Что делать после создания сервера

1. Запишите IP адрес
2. Запишите пароль root (или создайте SSH ключ)
3. Переходите к **Этапу 2** в `DEPLOY_SELECTEL.md`
