# 🎨 Варианты деплоя фронтенда (все бесплатно)

## Вариант 1: Vercel (рекомендуется) ✅

**Бесплатно для статических сайтов:**
- ✅ Expo web build = статический сайт
- ✅ Неограниченное количество деплоев
- ✅ 100GB трафика в месяц
- ✅ HTTPS из коробки
- ✅ CDN по всему миру
- ✅ Автоматические деплои при push

**Ограничения:**
- ⚠️ Serverless функции ограничены (но нам не нужны)
- ⚠️ Edge функции ограничены (но нам не нужны)

**Для нашего случая (статический сайт) - полностью бесплатно!**

---

## Вариант 2: Netlify (альтернатива)

**Бесплатно:**
- ✅ 100GB трафика в месяц
- ✅ Неограниченное количество деплоев
- ✅ HTTPS из коробки
- ✅ CDN

**Инструкция:**
1. https://netlify.com
2. Войдите через GitHub
3. New site from Git
4. Выберите репозиторий
5. Build command: `npm run web:build`
6. Publish directory: `.expo/web-build`

---

## Вариант 3: Cloudflare Pages (альтернатива)

**Бесплатно:**
- ✅ Неограниченный трафик
- ✅ Неограниченное количество деплоев
- ✅ HTTPS из коробки
- ✅ CDN

**Инструкция:**
1. https://pages.cloudflare.com
2. Войдите через GitHub
3. Create a project
4. Выберите репозиторий
5. Build command: `npm run web:build`
6. Build output directory: `.expo/web-build`

---

## Вариант 4: GitHub Pages (самый простой)

**Бесплатно:**
- ✅ Неограниченный трафик
- ✅ HTTPS из коробки
- ⚠️ Нужно настроить GitHub Actions для автоматической сборки

**Инструкция:**
1. Создайте файл `.github/workflows/deploy.yml`
2. Настройте автоматическую сборку и деплой
