# Анализ референс-дизайна: Prisma Template

## Референс
**URL:** https://ego-prisma-tmpl.framer.website/

## 1. Цветовая палитра

### Основные цвета референса:
- **Фон основной:** `rgb(0, 0, 0)` (черный)
- **Текст основной:** `rgb(255, 255, 255)` (белый)
- **Текст вторичный:** `rgba(255, 255, 255, 0.8)` (белый 80% прозрачности)
- **Акцент фиолетовый:** `rgb(156, 128, 244)` (#9c80f4)
- **Акцент фиолетовый темный:** `rgb(112, 52, 228)` (#7034e4)
- **Ссылки:** `rgb(0, 0, 238)` (синий)
- **Фон навигации:** `rgba(19, 19, 19, 0.72)` (темно-серый с прозрачностью)

### Градиенты:
- **Главный градиент (hero секция):**
  ```
  radial-gradient(200% 184% at 50% 5%, 
    rgb(0, 0, 0) 31.31%, 
    rgb(112, 52, 228) 47.50%, 
    rgb(156, 128, 244) 58.40%, 
    rgb(255, 255, 255) 85.77%)
  ```

### Корпоративные цвета (для замены):
- **White:** `#ffffff` (255/255/255)
- **WhiteSmoke:** `#f0f0f0` (240/240/240)
- **Pattens Blue:** `#cfe1ed` (207/225/237)
- **Spring Bud:** `#b6ff00` (182/255/0)
- **Medium Slate Blue:** `#7d51fe` (125/81/254)
- **Midnight Express:** `#242834` (36/40/52)

## 2. Типографика

### Шрифты:
1. **Urbanist** — для главных заголовков (H1)
   - Размер: 48px
   - Вес: 600 (semibold)
   - Line-height: 48px

2. **Lexend** — для заголовков H2, H3, H4
   - H2: 40px, weight 400, line-height 48px
   - H3: 32px, weight 400, line-height 35.2px
   - H4: 24px, weight 400, line-height 28.8px

3. **Inter** — для основного текста и навигации
   - Основной текст: 16px, weight 400
   - Навигация: 12px, weight 400
   - Подзаголовки: 18px, weight 400-500
   - Мелкий текст: 14px, weight 400-500

### Примеры использования:
- **H1 (Hero):** "One place for all your investing" — Urbanist 48px/600, белый
- **H2 (Секции):** "Working capital that just works flawless" — Lexend 40px/400, белый
- **H3 (Подзаголовки):** "Make informed* decisions..." — Lexend 32px/400, белый
- **Body:** Основной текст — Inter 16px/400, белый/80% прозрачности
- **Navigation:** Ссылки меню — Inter 12px/400, синий (#0000ee)

## 3. Компоненты и их стили

### Навигация (Navigation)
- **Фон:** Прозрачный `rgba(0, 0, 0, 0)`
- **Padding:** `16px 20px 0px`
- **Position:** `relative`
- **Ссылки навигации:**
  - Padding: `8px 12px`
  - Border-radius: `8px`
  - Цвет: `rgb(0, 0, 238)` (синий)
  - Font-size: `12px`
  - Height: `40px`

### Кнопки CTA

#### Основная кнопка (Primary CTA):
- **Текст:** "Buy Template", "Learn more"
- **Фон:** `rgb(255, 255, 255)` (белый)
- **Цвет текста:** `rgb(0, 0, 238)` (синий)
- **Padding:** `12px 24px`
- **Border-radius:** `64px` (очень скругленная)
- **Font-size:** `12px`
- **Font-weight:** `400`
- **Height:** `48px`
- **Display:** `flex`

#### Вторичная кнопка (Secondary):
- **Фон:** Прозрачный
- **Цвет текста:** `rgb(0, 0, 238)` (синий)
- **Padding:** `12px 24px`
- **Border-radius:** `100px`
- **Height:** `42px` или `48px`

### Секции (Sections)
- **Padding:** 
  - Большие секции: `160px 0px` или `120px 0px`
  - Средние секции: `80px 0px`
  - Малые секции: `80px 0px 32px` или `80px 0px 120px`
- **Margin:** `0px`
- **Фон:** Прозрачный (темный фон задается на уровне body/container)

### Карточки планов (Solution Cards)
- Структура: кнопка с внутренним контентом
- Содержат: заголовок (H3), подзаголовок (H4), описание, кнопку "Learn more"
- Изображения могут быть внутри карточек

### Блоки с иконками/фичами
- Заголовок H3: 24px, Lexend, weight 400
- Описание: Inter, 16-18px
- Расположение: сетка или список

## 4. Структура страницы

### Основные секции:
1. **Hero/Banner** — главный экран с заголовком и CTA
2. **About** — секция "О нас" с большим текстом
3. **Features** — блоки с фичами (Strategy, Protection, Growth)
4. **Solutions** — карточки с планами (Treasure, Crypto, Investments)
5. **Benefits** — преимущества (3 колонки)
6. **Stats/Numbers** — статистика с большими цифрами
7. **Reviews/Cases** — отзывы клиентов
8. **Blog** — статьи блога в виде карточек
9. **FAQ** — часто задаваемые вопросы (аккордеон)
10. **CTA Section** — форма подписки
11. **Footer** — футер с навигацией и контактами

## 5. Особенности дизайна

### Визуальные эффекты:
- **Темная тема:** Черный фон с белым текстом
- **Градиенты:** Радиальный градиент в hero-секции
- **Большие отступы:** Щедрые padding'и между секциями (80-160px)
- **Скругленные кнопки:** Очень большие border-radius (64px, 100px)
- **Минималистичная навигация:** Прозрачный фон, простые ссылки

### Анимации и интерактивность:
- Плавные переходы (предположительно)
- Hover-эффекты на кнопках и ссылках
- Аккордеон в FAQ секции

### Типографические особенности:
- Большие заголовки (40-48px)
- Разделение букв в некоторых заголовках (визуальный эффект)
- Использование специальных символов (*, #, =) в тексте для акцента

## 6. Адаптация под корпоративные цвета

### Рекомендации по замене:
1. **Черный фон** → `#242834` (Midnight Express)
2. **Фиолетовые акценты** → `#7d51fe` (Medium Slate Blue)
3. **Белый текст** → `#ffffff` (White)
4. **Светлые акценты** → `#b6ff00` (Spring Bud) для CTA
5. **Фон карточек** → `#f0f0f0` (WhiteSmoke) или `#cfe1ed` (Pattens Blue)
6. **Градиент:** Адаптировать под новые цвета:
   ```
   radial-gradient(200% 184% at 50% 5%, 
     #242834 31.31%, 
     #7d51fe 47.50%, 
     #cfe1ed 58.40%, 
     #ffffff 85.77%)
   ```
