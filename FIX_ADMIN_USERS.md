# 🔧 Исправление: Пропали пользователи в админке

## Проблема
В админке не отображаются пользователи.

## Возможные причины

### 1. Файл users.json пустой или не существует

Бэкенд использует файловое хранилище. Файл находится в:
```
backend/data/users.json
```

**Проверка:**
- Если файла нет - пользователи еще не регистрировались
- Если файл пустой `[]` - пользователей нет

### 2. Проблема с API запросом

**Проверьте:**
1. Откройте админку
2. Откройте консоль браузера (F12)
3. Перейдите в раздел "Пользователи"
4. Посмотрите ошибки в консоли

### 3. Проблема с авторизацией админа

**Проверьте:**
- Токен админа сохранен в localStorage
- API запрос идет с правильным токеном

## Решение

### Шаг 1: Проверить данные на бэкенде

```bash
# Проверить, есть ли пользователи
curl https://horeca-backend-6zl1.onrender.com/api/admin/users \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
```

### Шаг 2: Зарегистрировать тестового пользователя

1. Откройте основное приложение
2. Зарегистрируйтесь
3. Проверьте админку снова

### Шаг 3: Проверить логи Render

1. Откройте Render Dashboard
2. Выберите сервис `horeca-backend`
3. Перейдите в **Logs**
4. Проверьте ошибки при запросе `/api/admin/users`

## Если пользователей действительно нет

Это нормально, если:
- Приложение только что развернуто
- Еще никто не регистрировался
- Данные были очищены

**Решение:** Зарегистрируйте тестового пользователя через основное приложение.
