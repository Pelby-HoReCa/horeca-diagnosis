## ⚠️ Что нужно сделать

### 1. Добавить переменные окружения в Render

В Render Dashboard → ваш сервис `horeca-backend` → Environment Variables:

```
ADMIN_EMAIL=Sergo1289
ADMIN_PASSWORD=1289
```

После добавления Render автоматически перезапустит сервис.

### 2. Установить зависимости админки

```bash
cd admin
npm install
```

### 3. Настроить переменные окружения для админки

Создать `admin/.env.local`:
```
NEXT_PUBLIC_API_URL=https://horeca-backend-6zl1.onrender.com
```

### 4. Деплой админки на Vercel

1. Создать новый проект в Vercel
2. Указать корневую папку: `admin`
3. Build Command: `npm run build`
4. Output Directory: `.next`
5. Добавить переменную окружения:
   - `NEXT_PUBLIC_API_URL` = `https://horeca-backend-6zl1.onrender.com`
