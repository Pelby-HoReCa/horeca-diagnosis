### Логика работы

**Для web-версии:**
1. При нажатии на кнопку "Обновить приложение"
2. Показывается диалог подтверждения через `window.confirm()`
3. При подтверждении выполняется `window.location.reload()`
4. Страница перезагружается, получая последнюю версию с сервера

**Для мобильных (iOS/Android):**
1. Сохранена оригинальная логика через Expo Updates
2. Проверка обновлений через `Updates.checkForUpdateAsync()`
3. Загрузка и применение через `Updates.fetchUpdateAsync()` и `Updates.reloadAsync()`

### Преимущества решения

1. ✅ **Простота**: Для web достаточно перезагрузки страницы
2. ✅ **Надежность**: `window.confirm()` работает на всех браузерах
3. ✅ **Совместимость**: Не ломает существующую функциональность для мобильных
4. ✅ **UX**: Пользователь получает понятное сообщение и может отменить действие
