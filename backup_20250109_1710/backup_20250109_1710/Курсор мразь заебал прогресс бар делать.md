# Слепок приложения - Прогресс-бар на BlockResultsScreen

## Дата создания: $(date)

## Текущее состояние прогресс-бара

### Расположение
- Прогресс-бар находится **снаружи белого поля**, ниже его на 3 пикселя
- Расположен между белым полем (efficiencySection) и кнопкой "Продолжить диагностику"
- Имеет `zIndex: 100` для отображения поверх всех элементов

### Стили прогресс-бара
```typescript
progressBarContainer: {
  flexDirection: 'row',
  justifyContent: 'center',
  alignItems: 'center',
  marginTop: 0,
  marginBottom: 120,
  paddingHorizontal: spacing.md,
  paddingVertical: spacing.sm,
  minHeight: 20,
  width: '100%',
  backgroundColor: 'transparent',
  position: 'relative',
  zIndex: 100,
}
```

### Стили белого поля
```typescript
efficiencySection: {
  backgroundColor: palette.white,
  borderRadius: radii.md,
  paddingHorizontal: spacing.md,
  paddingTop: spacing.lg + 20,
  paddingBottom: spacing.md - 5 + 10 - 5 - 5,
  marginBottom: 3, // 3 пикселя отступ снизу для прогресс-бара
  marginTop: -4, // Поднято выше на 4 пикселя
  shadowColor: palette.midnightBlue,
  shadowOffset: { width: 0, height: 4 },
  shadowOpacity: 0.05,
  shadowRadius: 8,
  elevation: 3,
}
```

### Логика отображения точек
- **Пройденные блоки** (index < currentBlockIndex): зеленые точки (#38C793), размер 6px
- **Текущий блок** (index === currentBlockIndex): белая точка с зеленой обводкой, размер 10px
- **Непройденные блоки** (index > currentBlockIndex): серые точки (#C7D4E0), размер 6px

### SVG иконки
- `progress-dot-completed.svg` - для пройденных блоков
- `progress-dot-current.svg` - для текущего блока
- `progress-dot-incomplete.svg` - для непройденных блоков

### Структура компонента
Прогресс-бар рендерится после закрытия `contentContainer`, но перед кнопкой "Продолжить диагностику":

```tsx
</View> // закрытие contentContainer

{/* Прогресс-бар блоков - ниже белого поля на 3 пикселя */}
{DEFAULT_BLOCKS && DEFAULT_BLOCKS.length > 0 && (
  <View style={styles.progressBarContainer}>
    {/* точки */}
  </View>
)}

{/* Кнопка "Продолжить диагностику" */}
```
