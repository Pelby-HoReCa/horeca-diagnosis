- Спидометр серый (#E2E4E9) без секций

- Кнопка "Продолжить диагностику" показывает название следующего блока
- Спидометр с двумя секциями (зеленая/красная)
- Процент эффективности в спидометре
- Метрики с реальными значениями

### Логика навигации
1. RegisterScreen3 → "Начать диагностику" → NextBlockScreen (первый блок)
2. NextBlockScreen → "Продолжить диагностику" → DiagnosisQuestionsScreen (вопросы текущего блока)
3. DiagnosisQuestionsScreen → при завершении → BlockResultsScreen (результаты)
4. BlockResultsScreen → "Продолжить диагностику" → NextBlockScreen (следующий блок)
5. Цикл повторяется
