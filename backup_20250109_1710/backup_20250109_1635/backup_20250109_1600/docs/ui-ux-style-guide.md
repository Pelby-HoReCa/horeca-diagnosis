# UI/UX Style Guide (Draft)

## 1. Brand Palette

| Role | Hex | Usage |
|------|-----|-------|
| Primary Blue | `#112677` | Основные заголовки, акцентные фоны, иконки в навигации |
| Primary Orange | `#E84411` | CTA-кнопки, активные состояния, графические акценты |
| Midnight Blue | `#0B1A4A` | Бэкграунд модальных оверлеев, тёмные карточки |
| Sky Blue | `#3C5AFF` | Ховеры и вторичные акценты |
| Neutral Light (Gray 100) | `#F5F6FA` | Общий фон экранов |
| Neutral Medium (Gray 300) | `#D4D7E2` | Разделители, бордеры |
| Neutral Dark (Gray 600) | `#636984` | Подзаголовки, вторичный текст |
| Success Green | `#03A66A` | Положительные индикаторы |
| Warning Amber | `#FFAD1F` | Предупреждения |
| Error Red | `#FF4D57` | Ошибки | 

## 2. Typography

- **Заголовок H1**: 28 pt, semibold, основная синяя палитра
- **Заголовок H2**: 22 pt, semibold
- **Заголовок H3**: 18 pt, medium
- **Тело/карточки**: 16 pt, regular, цвет `Gray 600`
- **Подписи/чипы**: 14 pt, medium, верхний регистр
- **Вспомогательные**: 12 pt, medium, используется для подсказок/лейблов

Используем системный шрифт (SF Pro / Roboto) с указанными размерами. Для мобильных платформа можно подключить `Inter` или `SF Pro` через Expo fonts при необходимости.

## 3. Spacing & Layout

| Token | Value |
|-------|-------|
| `space-xxs` | 4 pt |
| `space-xs` | 8 pt |
| `space-sm` | 12 pt |
| `space-md` | 16 pt |
| `space-lg` | 24 pt |
| `space-xl` | 32 pt |
| `space-xxl` | 48 pt |

- Основные контейнеры: `padding-horizontal: space-md`, `padding-vertical: space-md`
- Карточки: внутренний паддинг `space-md`, отступ между карточками `space-md`
- Bottom sheets / модальные окна: `padding: space-lg`, радиус `24 pt`

## 4. Component Tokens

### Buttons

- **Primary CTA**: фон `Primary Orange`, текст белый, радиус `16 pt`, высота `52 pt`, тень 6 pt, состояние нажатия — затемнение до 90% непрозрачности.
- **Secondary**: фон белый, бордер `Primary Blue`, текст `Primary Blue`, высота `48 pt`, радиус `16 pt`.
- **Tertiary**: без фона, только текст `Primary Blue`, подчёркивание при ховере/нажатии.

### Inputs

- Высота `52 pt`, бордер `1 pt` (`Gray 300`), радиус `12 pt`, активное состояние — бордер `Primary Blue`, фон белый, тень 2 pt.
- Лейбл над полем (12 pt, medium, `Gray 600`), обязательные поля помечаются символом `*_` (без переноса).

### Cards & Containers

- Радиус `20 pt`, тень `0 4 16 rgba(17,38,119,0.08)`
- Заголовок H3, подзаголовки и теги соблюдают палитру
- Для списков задач/рекомендаций: в карточке оставить вертикальную полосу в цвете приоритета (4 pt)

### Navigation Tabs

- Bottom tab: активная иконка окрашивается в `Primary Blue`, подпись semibold, неактивные — `Gray 600`
- Верхние секции (например, табы задач) используют pill-кнопки с радиусом `999 pt` и индикацией цветом

### Charts & Data Visuals

- Бар-чарт: подписи над столбцами, минимальный высотный отступ 8 pt, цвета баров: `Primary Blue`, `Primary Orange`, `Warning Amber`, `Success Green` в зависимости от диапазона
- Дашборд-компаратор: пока не все блоки завершены — отображать плейсхолдер, иначе — реальное значение

## 5. States & Feedback

- **Loading**: семипрозрачный оверлей `rgba(17,38,119,0.12)` + индикатор
- **Empty states**: иконка (32 pt), заголовок H3, описание 14 pt `Gray 600`, CTA-кнопка вторичного типа
- **Errors**: баннер с фоном `Error Red` и белым текстом, иконка `alert-circle`
- **Success**: toasts с фоном `Success Green`

## 6. Accessibility & Interaction

- Цветовой контраст минимум 4.5:1 для текста
- Touch-target ≥ 48×48 pt
- Анимации — быстрые (150–200 ms), ease-in-out
- Фокусное состояние для клавиатурной навигации: яркая окантовка `Primary Orange`

## 7. Иконки

- Базироваться на Ionicons (Expo), толщина 1.5 pt
- В карточках и списках — размер 24 pt, в таб-баре — 22 pt

## 8. Иллюстрации/фото

- Профиль: круглые аватары, бордер `2 pt Primary Blue`
- Бэкграунды Splash/Auth: градиент от `Primary Blue` к `Midnight Blue`
