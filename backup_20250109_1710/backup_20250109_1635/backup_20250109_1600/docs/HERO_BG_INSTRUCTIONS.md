# Инструкция по использованию Hero Background Decoration

## Обзор

Созданы улучшенные версии фона для hero-секции в стиле референса PRISMA. Вместо плавного радиального градиента используются геометрические фигуры с резким переходом от фиолетового к черному.

## Доступные версии

### 1. **tilda-hero-bg-decoration.html** (ОСНОВНАЯ - РЕКОМЕНДУЕТСЯ)
   - Полная версия с несколькими геометрическими фигурами
   - Максимально похожа на референс PRISMA
   - Создает эффект глубины и объема
   - Использует 4 дополнительных элемента

### 2. **tilda-hero-bg-decoration-simple.html** (УПРОЩЕННАЯ)
   - Легкая версия с минимальным количеством элементов
   - Быстрее загружается
   - Использует только псевдоэлементы ::before и ::after
   - Подходит для слабых устройств

### 3. **tilda-hero-bg-decoration-improved.html** (АЛЬТЕРНАТИВНАЯ)
   - Альтернативный вариант с другим расположением фигур
   - Можно использовать для экспериментов

### 4. **tilda-hero-bg-decoration-prisma-style.html** (ЭКСПЕРИМЕНТАЛЬНАЯ)
   - Максимально детализированная версия
   - Больше геометрических элементов
   - Для случаев, когда нужен максимальный эффект

## Как использовать

1. Откройте файл с нужной версией
2. Скопируйте весь код (включая `<style>` и `<div>`)
3. В Тильде создайте Zero блок
4. Вставьте скопированный код в Zero блок
5. Убедитесь, что блок имеет `position: relative` (если нужно)
6. Проверьте `z-index` - фон должен быть за контентом

## Цвета

Используются корпоративные цвета:
- **Черный фон:** `#000000`
- **Фиолетовый:** `#7d51fe` (Medium Slate Blue) с различной прозрачностью

## Настройка

Если нужно изменить интенсивность фиолетового:
- Найдите `rgba(125, 81, 254, X)` где X - прозрачность (0.0 - 1.0)
- Увеличьте значение для более яркого эффекта
- Уменьшите для более темного фона

## Технические детали

- Используется `clip-path: polygon()` для создания угловатых форм
- `transform: rotate()` для поворота фигур
- `linear-gradient` и `radial-gradient` для плавных переходов внутри фигур
- Все элементы имеют `pointer-events: none` чтобы не мешать кликам

## Совместимость

- Современные браузеры (Chrome, Firefox, Safari, Edge)
- Поддержка `clip-path` обязательна
- Для старых браузеров можно использовать fallback с обычным градиентом

## Рекомендации

1. Начните с основной версии (`tilda-hero-bg-decoration.html`)
2. Если сайт тормозит - используйте упрощенную версию
3. Экспериментируйте с позициями фигур, меняя значения `top`, `left`, `right`, `bottom`
4. Можно менять углы поворота через `transform: rotate()`
