# Инструкция по установке прелоадера на Тильду

## 📁 Доступные файлы

1. **`tilda-preloader-embed.html`** - Основная версия (фиолетовый градиентный текст)
2. **`tilda-preloader-embed-variant2.html`** - Вариант 2 (темно-серый текст с квадратиком над Y)
3. **`tilda-preloader-embed-premium.html`** - Премиум версия (улучшенный 3D стеклянный эффект)

## 🚀 Как установить

### Шаг 1: Откройте редактор Тильды
1. Войдите в свой аккаунт Тильды
2. Откройте нужную страницу для редактирования

### Шаг 2: Добавьте блок HTML
1. Нажмите кнопку **"Добавить блок"** (обычно в левом верхнем углу)
2. Найдите и выберите блок **"HTML код"** или **"Code"**
3. Перетащите его в самое начало страницы (первым блоком)

### Шаг 3: Вставьте код
1. Откройте один из файлов прелоадера (рекомендуется `tilda-preloader-embed-premium.html`)
2. Скопируйте **весь код** из файла (Ctrl+C / Cmd+C)
3. Вставьте код в блок HTML в Тильде (Ctrl+V / Cmd+V)
4. Сохраните изменения

### Шаг 4: Проверка
1. Нажмите **"Предпросмотр"** или **"Опубликовать"**
2. Откройте страницу в новом окне
3. Прелоадер должен появиться на 5 секунд при загрузке страницы

## ⚙️ Настройки

### Изменить длительность прелоадера
Найдите в коде строку:
```javascript
}, 5000);  // 5000 = 5 секунд (в миллисекундах)
```
Измените `5000` на нужное значение:
- 3000 = 3 секунды
- 4000 = 4 секунды
- 5000 = 5 секунд (по умолчанию)
- 7000 = 7 секунд

### Изменить текст
Найдите в HTML части:
```html
<div class="royalty-logo-main" data-text="ROYALTY TEAMS">ROYALTY TEAMS</div>
<div class="royalty-logo-subtitle">ECO-SYSTEM FOR BUSINESS</div>
```
Измените текст на нужный.

### Изменить цвета
Все цвета заданы через CSS переменные и корпоративные цвета:
- `#242834` - Midnight Express (темный фон)
- `#7d51fe` - Medium Slate Blue (фиолетовый акцент)
- `#cfe1ed` - Pattens Blue (светло-голубой)
- `#b6ff00` - Spring Bud (зеленый акцент)
- `#ffffff` - White (белый)

## 🔧 Решение проблем

### Прелоадер не появляется
1. Убедитесь, что блок HTML находится **первым** на странице
2. Проверьте, что код вставлен полностью (включая `<style>` и `<script>`)
3. Очистите кеш браузера (Ctrl+Shift+R / Cmd+Shift+R)

### Прелоадер не исчезает
1. Проверьте консоль браузера на ошибки (F12)
2. Убедитесь, что JavaScript не заблокирован
3. Попробуйте уменьшить время до 3000ms для теста

### Конфликт со стилями Тильды
Если прелоадер конфликтует с другими стилями:
1. Добавьте `!important` к критическим стилям
2. Увеличьте z-index до `999999`
3. Используйте более специфичные селекторы

### Текст не отображается правильно
1. Убедитесь, что шрифт Inter доступен (он подключается автоматически)
2. Проверьте, что `-webkit-background-clip: text` поддерживается браузером
3. Для старых браузеров используйте вариант 2 (без градиента)

## 📱 Адаптивность

Прелоадер автоматически адаптируется под разные размеры экранов:
- **Десктоп:** Крупный текст (до 140px)
- **Планшет:** Средний текст (80-120px)
- **Мобильный:** Компактный текст (36-80px)

## 🎯 Рекомендации

1. **Используйте премиум версию** для лучшего визуального эффекта
2. **Размещайте блок HTML первым** на странице
3. **Тестируйте на разных устройствах** перед публикацией
4. **Оптимизируйте время** - 5 секунд оптимально, но можно уменьшить до 3-4 секунд

## 📞 Поддержка

3. Попробуйте другой вариант прелоадера
